# Actions_Plex
Actions that syncs the movies, shows and ratings between trakt and Plex (without needing a PlexPass or Trakt VIP subscription)

# Secrets

| Name                  | 说明                         |
| :-------------------: | ---------------------------- |
| `SET_ENV`             |   Plex Trakt 登录信息配置文件   |
| `SET_PYTRAKT`         |   Trakt API 信息配置文件       |
